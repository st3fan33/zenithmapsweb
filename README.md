# ZenithMaps - Romanian Mountains Mapping Website

Professional demo website for a Romanian mountain mapping business.

## Pages

- **index.html** - Homepage with hero, services overview, about, gallery preview, testimonials, features, and contact
- **services.html** - Detailed mountain range maps (Făgăraș, Bucegi, Retezat, Piatra Craiului, etc.) with pricing
- **about.html** - Company story, team, mission, partners, and statistics
- **gallery.html** - Photo gallery with filtering by mountain range
- **contact.html** - Contact form, location info, FAQ section

## Features

✅ Fully responsive design (mobile, tablet, desktop)
✅ Modern animations and transitions
✅ Romanian mountain-focused content
✅ Interactive elements (gallery filters, mobile menu, scroll effects)
✅ Professional color scheme matching the logo
✅ Contact form with validation
✅ Back-to-top button
✅ Scroll progress indicator
✅ SEO-friendly structure
